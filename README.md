# ShiftLabAuthRep


Тестовое задание для ШИФТ Lab: форма авторизации по номеру телефона и OTP-коду.

## Стек

- React
- TypeScript
- Vite
- CSS

## Возможности

- ввод номера телефона
- ввод только цифр
- валидация обязательного телефона
- ввод OTP-кода из 6 цифр
- состояние загрузки кнопок
- таймер повторной отправки кода
- mock-авторизация по коду `123456`

## Запуск

```bash
npm install
npm run dev
